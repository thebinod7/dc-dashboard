## Run Locally

Clone the project

```bash
  git clone git@github.com:dev-column/admin.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Add env variables

```bash
  Add `.env` file in project root
  Add REACT_APP_API_SERVER=http://localhost:7777

```

Start the server

```bash
  yarn start
```
